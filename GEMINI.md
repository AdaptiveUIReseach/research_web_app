# Project Instructions

- Follow `AGENTS.md` before editing code.
- This app uses Next.js 16.2.9 and React 19.2.4.
- Before changing Next.js APIs, routes, or conventions, read the relevant guide in `node_modules/next/dist/docs/` because this Next.js version may differ from older conventions.
- Prefer existing project patterns and keep changes scoped.

## Common Commands

- `npm run dev` starts the development server.
- `npm run build` verifies the production build.
- `npm run lint` runs ESLint.
